================================================================================
  ENDPOINTS IMPLEMENTED
================================================================================

PUBLIC (No Authentication):
  POST   /api/v1/auth/registrar        - Register new company + admin
  POST   /api/v1/auth/login            - Login (email + password)
  POST   /api/v1/auth/refresh          - Renew access token
  GET    /health                       - Health check

AUTHENTICATED (Require JWT):
  GET    /api/v1/auth/perfil           - Get user profile
  POST   /api/v1/auth/trocar-senha     - Change password
  GET    /api/v1/tenants/meu           - Get company data
  PUT    /api/v1/tenants/meu           - Update company (admin only)
  GET    /api/v1/usuarios              - List users in tenant
  POST   /api/v1/usuarios              - Invite user (admin/gerente only)
  DELETE /api/v1/usuarios/:id          - Deactivate user (admin only)

1. Install dependencies:
   yarn install

2. Copy environment template:
   cp .env.example .env.local

3. Start PostgreSQL + Kafka:
   docker-compose up -d

4. Run database migrations:
   yarn db:migrate

5. Seed test data:
   yarn db:seed

6. Start development server:
   yarn dev

7. Access API:
   http://localhost:3001/api/docs (Swagger)
